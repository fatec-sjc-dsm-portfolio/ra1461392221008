Beelieve

Segundo Semestre (2023-2)


Descrição do Projeto

O projeto Beelieve, desenvolvido em colaboração com a empresa SIATT no setor armamentista, visa facilitar a gestão de equipes multidisciplinares em projetos complexos. Utiliza metodologias como IPDT e WBS para assegurar uma gestão técnica e eficiente.

O sistema foi dividido em:

  1. Back-End: Java com Spring Boot.
  2. Front-End: React com Tailwind CSS.
  3. Banco de Dados: MySQL e MongoDB.

Contribuições Pessoais

Atuei principalmente no Back-End, contribuindo em diversas frentes:

  • Modelagem de bancos de dados SQL (MySQL) e NoSQL (MongoDB).
  • Planejamento da arquitetura do Back-End seguindo padrões REST e POO.
  • Programação com Java Spring Boot, com foco em eficiência e segurança.
  • Implementação do MongoDB e Spring Security.
  • Integração de ferramenta de criptografia via hashing para senhas.
  • Criação e execução de casos de teste funcionais e de integração.

Aprendizados e Experiência

Hard Skills
  • Modelagem de bancos de dados relacionais (SQL) e não relacionais (MongoDB).
  • Desenvolvimento Back-End: Java Spring Boot para aplicações escaláveis.
  • Implementação de APIs RESTful.
  • Aplicação de princípios da Programação Orientada a Objetos.
  • Segurança de aplicações com Spring Security e hashing de senhas.
Soft Skills
  • Colaboração efetiva com equipe multidisciplinar.
  • Comunicação clara com stakeholders e colegas.
  • Adaptabilidade diante de mudanças no escopo do projeto.
  • Comprometimento com a entrega eficiente e segura.
  • Resolução de problemas técnicos de forma proativa.

Esse projeto foi essencial para consolidar meus conhecimentos técnicos e aprimorar minha atuação colaborativa em equipes ágeis e exigentes.
