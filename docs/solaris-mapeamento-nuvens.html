Solaris - Mapeamento de Nuvens e Sombras de Nuvens

Quinto Semestre (2024 - 2)

Parceria com a empresa Visiona

Descrição do Projeto

O projeto tem como objetivo identificar automaticamente nuvens e suas sombras em imagens de satélite geradas pelo sensor WPM do satélite CBERS-4A. A solução integra componentes de Inteligência Artificial com dados obtidos via API do INPE.

As principais funcionalidades incluem:

  • Coleta de imagens de satélite via API do INPE.
  • Processamento automatizado com algoritmos de IA.
  • Geração de máscaras de nuvens e sombras.
  • Exibição dos resultados em plataformas Web e Mobile.

Contribuições Pessoais

Atuei como parte do time de desenvolvimento com foco principal no componente de Inteligência Artificial:

  • Criação do serviço de IA responsável pelo reconhecimento de nuvens e sombras.
  • Treinamento do modelo de IA com base nas imagens do sensor WPM.
  • Desenvolvimento de um sistema de anotação para compor e organizar o dataset.
  • Anotação manual das imagens para garantir a qualidade do conjunto de dados.

Essas contribuições foram fundamentais para a construção de uma solução confiável e eficiente.


Aprendizados e Experiência

  • Implementação de IA aplicada a imagens de satélite.
  • Manipulação de APIs externas (INPE) para coleta de dados.
  • Construção de datasets anotados manualmente para treinamento supervisionado.
  • Desenvolvimento em equipe em um projeto multidisciplinar com empresa parceira.
