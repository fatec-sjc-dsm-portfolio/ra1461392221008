Certificações

Certified Tester Foundation Level

A certificação ISTQB® Certified Tester Foundation Level (CTFL) é essencial para adquirir conhecimentos básicos em testes, aplicáveis a cenários do mundo real.

Agile Tester (CTFL-AT)

A Agile Tester (CTFL-AT) fornece as principais habilidades de teste necessárias para contribuir com sucesso para um projeto Ágil, seguindo os princípios do desenvolvimento ágil de software, conforme descrito no Manifesto Ágil.

Acceptance Testing (CT-AcT)

A Acceptance Testing (CT-AcT), é a certificação do ISTQB destinada a qualquer pessoa envolvida em teste de software que deseja ampliar seus conhecimentos em testes de aceite ou qualquer pessoa que deseja iniciar uma carreira de especialista em testes de aceite.
