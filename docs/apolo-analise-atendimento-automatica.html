Apolo - Análise Automatizada de Atendimentos com PLN

Sexto Semestre (2025 - 1)

Parceria com a empresa IT-01

Descrição do Projeto

O Projeto Apolo tem como objetivo automatizar a extração e análise de dados provenientes de atendimentos (como chatbots, CRMs e assistentes virtuais), utilizando técnicas de Processamento de Linguagem Natural (PLN).

Entre as funcionalidades desenvolvidas, destacam-se:

  • Extração de informações relevantes das interações.
  • Classificação automática de chamados (reclamações, dúvidas, solicitações).
  • Sumarização de conversas e geração de insights.
  • Dashboards interativos com métricas.
  • Anonimização de dados conforme a LGPD.
  • Busca semântica e análise de sentimentos.
  • Arquitetura escalável em nuvem.
  • Gerenciamento de usuários com diferentes níveis de acesso.

Contribuições Pessoais

Minhas principais contribuições técnicas e de liderança neste projeto incluíram:

  • Criação e implementação de diversos micro-serviços de IA.
  • Desenvolvimento dos módulos de:
    • Sumarização automática de conversas.
    • Busca semântica baseada em embeddings.
    • Classificação de sentimentos e tipos de documentos.
    • Anonimização de dados sensíveis para conformidade com a LGPD.
  • Treinamento dos modelos de linguagem para tarefas específicas do projeto.

Além disso, exerci o papel de Product Owner (PO):

  • Responsável pelo levantamento de requisitos diretamente com os stakeholders.
  • Transmissão clara da visão do projeto para toda a equipe de desenvolvimento.
  • Organização e priorização do backlog de funcionalidades.

Aprendizados e Experiência

  • Desenvolvimento de aplicações baseadas em PLN para contextos empresariais.
  • Integração de micro-serviços com arquitetura escalável em nuvem.
  • Prática de liderança de equipe e comunicação entre áreas técnicas e de negócio.
  • Aplicação da LGPD no desenvolvimento de software com dados sensíveis.
  • Uso de metodologias ágeis com foco em entregas contínuas de valor.
