Nimbus - Sistema de Coleta de Dados de Estações Meteorológicas

Quarto Semestre (2024-1)

Parceiro Acadêmico: Tecsus


Descrição do Projeto

Este projeto, desenvolvido pelos alunos do 4º semestre da FATEC Prof Jessen Vidal em parceria com a empresa Tecsus, tem como objetivo a coleta e visualização de dados meteorológicos obtidos através de sensores acoplados em estações distribuídas geograficamente.

O sistema disponibiliza esses dados em dashboards intuitivos, informando a população local e promovendo a conscientização sobre a importância do monitoramento climático.

Utilizamos uma arquitetura de microserviços escalável, combinando tecnologias modernas como Node.js com TypeScript, bancos de dados PostgreSQL e MongoDB, React com Next.js no front-end, e Tailwind CSS para estilização.


Contribuições Pessoais

Minhas principais contribuições no projeto foram:

Back-End
  • Planejamento da arquitetura de microserviços.
  • Desenvolvimento do microserviço de tratamento de dados meteorológicos.
  • Implementação de regras de negócio para geração de alertas.
  • Integração com bancos de dados PostgreSQL e MongoDB.
Integração e API
  • Desenvolvimento de API para comunicação entre o front-end e o back-end.
  • Criação de operações CRUD para alertas, usuários e estações meteorológicas.
DevOps
  • Configuração de pipelines automatizados de CI/CD.
  • Integração de testes unitários e end-to-end com Cypress.
