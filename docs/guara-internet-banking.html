Guará Internet Banking

Primeiro Semestre (2022-2)


Descrição do Projeto

O projeto Guará Internet Banking consistiu no desenvolvimento de um sistema para gestão de operações bancárias essenciais, tais como:

  • Depósitos
  • Saques
  • Transferências entre usuários

O sistema foi dividido em:

  1. Back-End: Desenvolvido em Python, utilizando o microframework Flask.
  2. Front-End: Implementado com Jinja2.
  3. Banco de Dados: Gerenciado com MySQL.

Contribuições Pessoais

Durante o projeto, desempenhei diversas funções no time de desenvolvimento, com destaque para:

Back-End
  • Estruturação das rotas no Flask para gerenciar funcionalidades como depósitos, saques e transferências.
  • Consultas ao banco de dados MySQL, garantindo a consistência dos dados.
Front-End
  • Organização dos arquivos HTML para a interface de usuário.
  • Integração dos dados dinâmicos do Back-End com o Front-End usando Jinja2.

Aprendizados e Experiência

Este projeto foi minha introdução ao desenvolvimento ágil e colaborativo, no qual aprendi a:

  • Trabalhar de forma proativa e autogerenciada em equipe.
  • Integrar Front-End e Back-End de forma eficiente.
  • Estruturar APIs com Flask e manipular bancos de dados MySQL.
  • Utilizar metodologias ágeis para planejamento e entregas.

Essa experiência foi essencial para o meu desenvolvimento técnico e profissional.
